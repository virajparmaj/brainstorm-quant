# Quant Project Brainstorm Repository

This repository is a structured space to explore, brainstorm, and progressively build quantitative research and ML-based trading strategies.

## Objective
Create a portfolio of high-impact quant research ideas and implementation pipelines that can be extended into:
- Trading/algo development
- Research papers
- Fellowship applications
- Hedge fund portfolio case studies

## Core Areas
1. **Basket Trading Optimization using Bayesian Optimization**
2. **Smart Predict-then-Optimize (SPO) Portfolio Models**
3. **Derivative Hedging using Reinforcement Learning**
